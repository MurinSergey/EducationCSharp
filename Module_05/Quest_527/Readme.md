## Задание 5.2.7
Изменить программу задания 5.2.2:
* Передать в метод возраст
* Вывести возраст при запросе цвета
* Сделать перенос строки с помощью Environment.NewLine

Обратите внимание: литерал "\n" работает для операционной системы Windows. 
Для работы без привязки к конкретной операционной системе необходимо использовать Environment.NewLine.
