## Финальное задание модуля 04
Необходимо разработать методику анкетирования пользователей. 
Требуется сохранять информацию: 
* об имени пользователя, string
* фамилии, string
* логине, string
* длине логина, int
* наличии/отсутствии у пользователя питомца, bool
* возрасте пользователя, int
* трех любимых цветах пользователя. string[3]

### Задание 4.5.1
Задайте кортеж User, который описывает необходимую информацию.

### Задание 4.5.2
Заполните кортеж User значениями с клавиатуры для полей: имя пользователя, фамилия, логин.

### Задание 4.5.3
Вычислите длину логина пользователя.

### Задание 4.5.4
Напишите условие, которое устанавливает значение ИСТИНА в поле наличие/отсутствие животных, если пользователь вводит "Да", и ЛОЖЬ при любом другом варианте.

### Задание 4.5.5
Напишите код для ввода возраста пользователя и трех его любимых цветов.
